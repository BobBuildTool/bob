Welcome to Bob's documentation!
===============================

.. image:: /BOB_Logo.png
   :target: https://bobbuildtool.github.io/

To get a glimpse on Bob start with the :doc:`Tutorials <tutorial/index>` or have
a look at the `website <https://bobbuildtool.github.io/>`_. For a comprehensive
reference on Bob's usage and features have a look at the :doc:`User manual
<manual/index>`.

There is also a `mini cheat sheet <_static/bob_cheatsheet.pdf>`_ with the most
common used commands, options and keywords.
